# FileEncryptor
AES-GCM file encryption tool - secure, efficient, with multi-strength options and large file support

## .NET 6.0 Simple AES-256-GCM File Encryptor

### Information:

- Runs on Windows x86/x64 (Winforms Application){need to install .NET 6.0}
- Uses AES-256-GCM to Encrypt/Decrypt files
- Encryption key derived from 300,000 iterations of Argon2id with salt
- Uses SHA-256 checksum to verify file integrity
- Supports all file formats with automatic extension preservation
